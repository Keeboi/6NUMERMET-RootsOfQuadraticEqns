# Roots of Quadratic Equations
A take home programming activity for the subject 6NUMERMET. 

# Project Requirements
Implement the following numerical methods for getting the roots of quadratic equations:
1. Bisection
2. Newton Rapson
3. Secant
4. Modified Secant
